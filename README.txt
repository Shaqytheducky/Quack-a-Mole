#Quack-a-Mole

Whack-a-mole game in HTML, CSS, and JavaScript...

## 🕹️ How to play
- Whack the mole to save time
- Avoid Shaqy and the eggs to not waste seconds
- The game lasts 60 seconds

## 🔧 Technologies
- HTML5
- CSS3
- JavaScript
- Custom 2D graphics
